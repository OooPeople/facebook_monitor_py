ntfy 說明

手機同步通知設定。

ntfy 是一個簡單能讓手機收到通知訊息的 app。

建議步驟

  1. 在手機上安裝 ntfy App。
  2. 在 App 內按 +，輸入 topic，例如 my-facebook-alerts。
  3. 建議使用英文字母、數字、減號或底線。
  4. 回到 Facebook Monitor 的通知設定。
  5. 勾選 ntfy 通道。
  6. 在 ntfy topic 輸入完全相同的 topic。
  7. 保存設定後，下一次符合關鍵字的新項目會透過 ntfy 通知。

Discord Webhook 說明

Discord 頻道通知設定。

這個設定可以讓通知訊息發送到 Discord 的頻道。

建議步驟

  1. 在 Discord 選擇目標頻道，進入「編輯頻道」。
  2. 點選「整合」→「Webhooks」→「新 Webhook」。
  3. 複製 Webhook URL。
  4. 回到 Facebook Monitor 的通知設定。
  5. 勾選 Discord Webhook 通道。
  6. 在 Discord Webhook URL 貼上剛剛複製的網址。
  7. 保存設定後，下一次符合關鍵字的新項目會透過 Discord 通知。
